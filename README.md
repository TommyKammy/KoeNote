  KoeNote

日本語の音声ファイルを、ローカルPC上で文字起こし・推敲レビュー・編集・出力できる Windows アプリです。

![KoeNote main screen](docs/images/koenote-main.png)

## KoeNote でできること

- 音声ファイルを登録して文字起こしできます
- 文字起こし結果を確認しながら音声を再生できます
- 推敲レビューで、より自然な文章への修正候補を確認できます
- 話者名や文字起こし本文を編集できます
- TXT / JSON / SRT / DOCX 形式で出力できます
- 音声データや文字起こしデータは、基本的に利用者のPC内で扱います

## はじめて使う流れ

1. KoeNote を起動します
2. ヘッダー左上の **音声追加** を押します
3. 文字起こししたい音声ファイルを選択します
4. ジョブ一覧に音声が登録されたら **実行** を押します
5. 文字起こしが完了したら、中央の文字起こし欄で内容を確認します
6. 必要に応じて、右側の推敲レビュー候補を確認します
7. 必要な形式を選んで出力します

## 画面の見方

### ヘッダー

画面上部には、よく使う操作があります。

- **音声追加**: 音声ファイルを登録します
- **実行**: 選択中のジョブを処理します
- **中止**: 実行中の処理を止めます
- **TXT / JSON / SRT / DOCX**: 出力形式を選びます
- **タイムスタンプ**: 出力に時刻情報を含めます
- **設定 / セットアップ / モデル / ログ**: 環境やモデル、ログを確認します

### ジョブ一覧

左側には、登録した音声ファイルが表示されます。

- 登録直後のジョブは **登録済み / 0%** と表示されます
- 実行中は進捗が表示されます
- 不要なジョブは削除できます

### 文字起こし

中央には、文字起こし結果が表示されます。

- 音声の再生位置に合わせて内容を確認できます
- セグメントを選択して本文を編集できます
- 自動スクロールを有効にすると、再生位置に近いセグメントへ追従します

### 推敲レビュー

右側には、推敲候補が表示されます。

- ASR結果が十分自然な場合、候補は表示されないことがあります
- 候補を採用・却下しながら本文を整えられます
- 推敲ステージは処理ステージのアイコンから ON / OFF を切り替えできます

### 処理ステージと再生バー

画面下部には、処理状況と音声再生バーがあります。

- **音声変換**
- **ASR**
- **推敲**
- **レビュー**

各ステージの状態を見ながら、音声を再生・停止・シークできます。

## セットアップ

初回利用時は、ヘッダーの **セットアップ** から環境確認を行います。

主に確認する項目は次のとおりです。

- 保存先
- ASRモデル
- 推敲LLM
- ライセンス確認
- モデル導入状態

すべてのチェックが完了すると、音声処理を実行できるようになります。

## モデル管理

ヘッダーの **モデル** から、文字起こしや推敲に使うモデルを確認できます。

- インストール済みモデルの確認
- 使用するモデルの切り替え
- モデルファイルの削除
- ライセンス確認

モデルファイルは容量が大きい場合があります。不要なモデルは削除することで、PCの空き容量を増やせます。

## データの保存場所

KoeNote のデータは、主に以下の場所に保存されます。

```text
%APPDATA%\KoeNote
%LOCALAPPDATA%\KoeNote
```

ログはアプリ内の **ログ** 画面から確認できます。

## 出力形式

KoeNote は以下の形式で出力できます。

- TXT
- JSON
- SRT
- DOCX

字幕用途では SRT、文章として確認・共有する場合は TXT または DOCX が便利です。
