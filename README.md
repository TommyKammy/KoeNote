# KoeNote

KoeNote is an experimental Windows desktop app for fully local Japanese ASR and review-assisted transcript correction.

## Development

- Runtime: .NET 11 preview
- UI: WPF
- Target reference machine: Windows 11 + GeForce RTX 3060 12GB
- Current development host: Windows 11 + GeForce RTX 3090
