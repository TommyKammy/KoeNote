# Phase 11 - Beta Operation

Phase 11 is for operating the beta, not adding broad new product areas.

## Goal

Validate KoeNote with real local transcription workflows and convert feedback into a v1.0 fix list.

## Scope

- test 10, 30, and 60 minute recordings
- collect recurring misrecognition patterns
- evaluate Correction Memory usefulness
- track failed exports and runtime failures
- review UX friction around draft selection and undo
- produce v1.0 must-fix list

## Completion Criteria

- Beta feedback is grouped by severity and workflow.
- Memory-derived candidate usefulness is measured.
- Long-audio runtime behavior is understood.
- v1.0 scope is explicitly accepted or cut.
